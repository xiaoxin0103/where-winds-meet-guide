## ⚔️ Escape from Tarkov Crossover — "渡尘墟" (Dust Ruins)

This is the big one. A **PvPvE extraction mode** set in the Hexi desert, built in collaboration with Battlestate Games (Escape from Tarkov).

### Core Mechanics

| Mechanic | Detail |
|----------|--------|
| **Mode Type** | PvPvE Extraction |
| **Team Size** | Solo or 3-player squads |
| **Combat** | **Melee weapons only** — no guns, closer engagements |
| **Economy** | Independent "金宝钞" currency — loot, extract, upgrade |
| **Collectibles** | 150+ items across 5 rarity tiers (White → Red) |

### How It Works

```
1. DEPLOY → Enter the Hexi desert map with your gear
2. LOOT → Search buildings, defeat enemies, find treasure
3. SURVIVE → Other players are hunting the same loot
4. EXTRACT → Reach an extraction point before time runs out
5. UPGRADE → Spend 金宝钞 on better gear for next run
```

### Unique Features

| Feature | Detail |
|---------|--------|
| **Disguise System** | Transform into objects or animals to avoid detection |
| **Assassination Moves** | Silent takedowns from stealth |
| **Limited Extraction** | Not everyone gets out — extraction slots are limited |
| **Safe-Box** | Protect your most valuable loot if you die |
| **Treasure Maps** | Find maps leading to hidden caches |
| **Random Events** | Dynamic events change each run |

### Loot Tiers

| Tier | Color | Examples |
|------|-------|----------|
| 1 | ⬜ White | Common materials, basic consumables |
| 2 | 🟩 Green | Uncommon gear, crafting ingredients |
| 3 | 🟦 Blue | Rare weapons, skill scrolls |
| 4 | 🟪 Purple | Epic gear pieces, rare treasures |
| **5** | **🟥 Red** | **Artifact collectibles — unlock exclusive cosmetics** |

<div class="callout tip">
<div class="callout-title">💡 Extraction Strategy</div>
Don't get greedy. Red-tier artifacts are worth extracting immediately. One successful extraction with a Red item > three failed runs with full bags.
</div>

## Best Builds for Dust Ruins (Extraction Mode)

Since the mode is **melee-only, close-quarters, and stealth-focused**, these weapon types will excel:

| Weapon | Why It's Good |
|--------|---------------|
| **Infernal Twinblades** | Fast combos, multiple dashes — best for hit-and-run |
