# 🛡️ Best Gear Sets & Stats Optimization (v1.7)

What gear to farm, which sets to combine, and how to prioritize your stats for maximum performance.

---

## Gear System Basics

- **Weapons:** 2 slots (main + offhand)
- **Armor:** Full set bonus at 4 pieces
- **Accessories:** Disc + Pendant (also contribute to set bonuses)
- **Gear Sets:** 2-piece and 4-piece bonuses. Mix across weapon/armor/accessory slots.
- **Substats:** Weapons gain random substats starting Tier 31, up to 4 slots by Tier 56.

## Stat Priority by Build

### Melee DPS (Sword / Spear)
```
1. Power
2. Affinity Rate
3. Bellstrike Attack / Min-Max Physical Attack
4. Critical Rate
```

### Burst DPS (Twinblades / Rope Dart)
```
1. Bamboocut Attack
2. Critical Rate
3. Agility
4. Physical Attack
```

### Tank (Thundercry Blade / Stormbreaker Spear)
```
1. Max HP / Body
2. Physical Defense
3. Affinity Rate
4. Critical Rate / Crit DMG (secondary)
```

### Healer (Fan / Umbrella)
```
1. Critical Rate          ← Procs Ivorybloom
2. Agility                ← Double value: Crit Rate + Min Phys Attack
3. Silkbind Attack        ← Healing scaling
4. Defense                ← Stay alive to heal
```

## Optimization Priority (Most to Least Important)

```
1. Inner Ways (talent nodes)     ← Mechanical breakpoints
2. Breakthroughs (cultivation)    ← Unlock weapon grades
3. Refine Arts (skill upgrades)   ← Daily Martial Points
4. Gear (sets + stats)            ← Amplifies an already-functioning build
```

> **Golden rule:** Gear amplifies a build that already works. Don't farm gear with incomplete talent nodes.

---

## Pro Tips

- **Attunements matter in endgame** — Physical Penetration, Formless Penetration, and Martial Art Skill DMG Boost can significantly increase output.
- **Mix 2-piece bonuses** while farming full 4-piece sets.
- **Substat rolls are random** — don't stress perfect rolls until Tier 56+.
- **Save upgrade materials** for your main weapon pair. Don't upgrade everything.
